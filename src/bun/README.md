
# Bun

Bun is a fast JavaScript runtime. [bun.sh](https://bun.sh)

## Example Usage

### Latest version
```json
"features": {
    "ghcr.io/raine-works/devcontainers-features/bun:1": {}
}
```

### Using a specific version
```json
"features": {
    "ghcr.io/raine-works/devcontainers-features/bun:1": {
        "version": "1.1.15"
    }
}
```
